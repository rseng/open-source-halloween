Trick or Treating, Open Source Style!

What's an easy way to have fun, and celebrate an open source Halloween? Open source trick or treating, of course! The candy here is our trick or treat bag that we've discovered across the open source landscape. You can participate here too! How does it work?

  • Generate a candy for your repository with the candy-generator
  • Take a screenshot of your candy, tight around the edges
  • Save it somewhere in your repository named "open-source-halloween-2021.png"
And that's it! The interface here runs a nightly job to search for these files, meaning that if GitHub has indexed it, it should show up. If you run into any issues with the generator please open an issue, or even better, a pull request with your suggested changes.
